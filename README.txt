This is a spotlight effect with interactable backgorund using JS and CSS.
The interactable HTML body must be put in the div with class and id as "spotlihtBody".

To change the spot light size --spotlightRadius in the CSS file must be changed in pixels.
Also in the JS file the variable spotlightRadius must be set accordingly.
